# Sublime 列对齐功能

1. Ctrl + A （全选）
2. Ctrl + Shift + L （进入列模式）
3. 按方向键（右键），使光标置于每一行末尾，解除全选状态
4. 再按 Ctrl + 方向键（左键），使光标置于第二列数据的开头
5. 最后 Ctrl + K（Alignment插件的改键，原来的ctrl+alt+a，这个组合键可以说和很多软件都有热键冲突，果断更改），使第二行数据对齐！
