### 3.Tip:

Git 获取指定历史版本源代码

1.bash 下执行：
``` bash
git clone http://xxx/xx.git
```
2.bash 下执行：cd 进入clone的项目目录
3.在 git 仓库查看分支的 SHA
4.bash 下执行对应版本的 SHA：
``` bash
git checkout fd9268a 
```
